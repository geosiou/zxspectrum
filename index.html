WELCOME!!!

This is my page about various tips and info for the Sinclair ZX Spectrum computer

I have found an interesting way to implement 2 or even 3 fire buttons (and even up to 10!!! - but more on this later!) support in games!!!

First of all we need a joystick interface to provide support for that and then an easy way to make games take advantage of this feature since games were never designed to support more than 1 fire button...

This joystick interface is the very versatile K-Mouse Turbo 2011 from George Velesoft! It is both a joystick interface which does support up to 3 fire buttons and a Kempston mouse compatible interface!

MOUSE discription

As a Kempston mouse interface it gives you the option to connect an older Amiga mouse (both 2 and 3 buttons mice supported) with a DB9 connector or a modern PS/2 compatible where it supports the wheel too. Modern USB mice work too (with a USB to PS/2 adapter) as long as the USB mouse supports the PS/2 protocol.

That 3rd mouse button / wheel support gives extra functions in comparison to the original Kempston mouse such as changing cursor speed setting and/or swapping left and right button function (and make Artist II work as Art Studio...).

JOYSTICK discription

As a joystick interface it has a standard DB9 connector and supports 4 interface modes with 3 fire buttons each:

1. Kempston mode

2. Sinclair ZX Interface 2 (port 1) enhanced mode - the standard 6, 7, 8, 9, 0, with extra fire buttons Z and X!

All are sensed as keypresses by the computer:
3. Fuller mode
4. Q, W, E, R, T, C, V mode. All are sensed as keypresses by the computer.

The most interesting modes are 2 and 4 since the "movements" are sensed as keypresses. That way you can just use the game's "redefine keys" option and then move your joystick and press its fire buttons to register them to the corresponding keypresses. For modes 1 and 3 you would need modifications in game's code to read the extra 2 fire buttons (instructions are given in the K MOUSE TURBO 2011 manual)...

If you are wiring a custom made joystick/joypad or modifying a ready made one, the above table is all you need!

Ready made joysticks/joypads which support 2 fire buttons without modifications are:

  • SEGA MASTER SYSTEM controller. A safe and ready solution with Fire 1 & Fire 2 buttons!

  • The SEGA GENESIS / MEGA DRIVE 3 button controller can also be used and it defaults the Fire 1 in its B button and the Fire 2 in its C button. Some say that it is dangerous to use a SEGA GENESIS/MEGA DRIVE with an retro computer (it does destroy Commodores), but as far as ZX Spectrum is concerned it is a matter of the joystick interface used, if it can handle the voltage that SEGA GENESIS/MEGA DRIVE returns to it and I think K-MOUSE TURBO 2011 can handle it well.
